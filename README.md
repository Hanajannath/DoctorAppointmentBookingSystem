Doctor Appointment Booking System (MediFlow)
MediFlow is a web-based Doctor Appointment Booking System built using Django.
It allows patients to book appointments easily, doctors to manage schedules, and admins to control the system â€” all in one platform.

Features:
-> Admin
 -Admin login
 -Add, edit, and block doctors
 -Manage doctor specializations
 -View all appointments

-> Doctor
 -Doctor login
 -Update profile 
 -Set available days and time slots
 -View appointment requests

-> Patient
 -Patient registration & login
 -Browse doctors by specialization
 -Book appointments
 -View appointment status

Tech Stack
-Backend: Python, Django
-Frontend: HTML, CSS, Bootstrap
-Database: SQLite

Deployment: PythonAnywhere
